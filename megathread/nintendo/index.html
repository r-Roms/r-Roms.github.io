  • .3ds/.cia Decryption Instructions:

If Citra says the roms are encrypted, use either download Batch CIA-3DS Decryptor to decrypt.

Or add the following AES keys to CitraAES Keys for Citra, Pastebin Link Click download on Pastebin to download the text in this file as a .txt file.

Windows installation instructions Add the downloaded pastebin text file to C:/Users/"your_user_name"/AppData/Roaming/Citra/sysdata make sure it is named aes_keys.txt

Linux/MacOS Installation instructions Add the downloaded pastebin text file to ~/.local/share/citra-emu/sysdata make sure it is named aes_keys.txt

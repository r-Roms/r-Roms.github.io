# Welcome to the /r/ROMS megathread, preserving classic games since 2018!

Welcome to the official repository for the /r/Roms Megathread. This is a curated collection of links to safely backup and preserve video game history.

## 🚀 Quick Start

If you are looking for games right now, please visit our [Official Megathread Website](https://r-roms.github.io/).


## 🛠️ Getting Started

### 1. How to Download

Most files are hosted on **Archive.org** or **Myrient**.

 - **For Archive.org:** We highly recommend creating a free account to bypass speed caps.

 - **Batch Downloading:** Use JDownloader 2 to download entire directories at once.

### 2. Extracting Files

If you encounter compressed files, use the following tools:

- **Windows:** 7-Zip (Recommended)

- **macOS:** The Unarchiver

- **Linux:** `p7zip`

## 🤝 Contributing & Support

- **Report Broken Links:** Please open an [Issue] or post in the /r/Roms subreddit.

- **Requesting Games:** Check the megathread thoroughly before requesting on the sub.

- **Safety:** All links provided here are vetted by the community. Always ensure you are using an adblocker (like **uBlock Origin**).

## 🛡️ Disclaimer

This repository is for **educational and preservation purposes only.** We do not host any copyrighted files on GitHub's servers; we only provide a directory of links to external archival sites. Users are responsible for complying with their local laws.
